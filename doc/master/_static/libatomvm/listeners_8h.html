Detailed Description

Common code for port listeners.

This header defines convenient common functions to implement listeners, and should be included in platform's sys.c.

Before including this file, define listener_event_t which represent a selectable event, as well as EventListener, which should have a list head member called listeners_list_head and a handler member called handler.

On a platform using select(3) with file descriptors, this typically is done by creating a platform_sys.h header with:

#include "sys.h"
typedef int listener_event_t;
struct EventListener
{
struct ListHead listeners_list_head;
event_handler_t handler;
listener_event_t fd;
};
